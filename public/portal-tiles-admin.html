Portal Tile Admin

Per-community control over which homeowner-portal tiles are visible, coming soon, or hidden.
← trustEd

One grid for every community × every tile

Each row is a community. The first toggle is the master kill-switch (Active = homeowners can log in; off = portal blocked entirely). The remaining columns are individual tiles — pick Live when the data/feature is ready, Coming Soon when it's planned, or Hidden for tiles that don't apply (no pool, no clubhouse, etc). Changes save automatically.

Live — tile shows and links to working page Coming Soon — tile shows but with placeholder copy Hidden — tile not rendered for this community
